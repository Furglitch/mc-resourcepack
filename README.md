# Furglitch's Minecraft Resource Packs

Various resource packs for Minecraft Java Edition created and maintained by Furglitch.

## Resource Packs Included

### Nitpicks
Small tweaks for various, insignificant mod issues

### Noticably Suspicious
Makes suspicious blocks more noticable, but not too cheaty.

### Ping Wheel - Portal 2 Sound
Replaces the default sound from **LukenSkyne's *Ping Wheel*** mod with the ***Portal 2 co-op*** ping sound

### Pretty Pretty Pipes
Brings the ***Pretty Pipez*** style to **Ellpack's *Pretty Pipes*** mod

### Pretty Pretty Pipes: Mekanism
Brings the ***Pretty Pipez: Mekanism Edition*** style to **Ellpack's *Pretty Pipes*** mod

### Utilitarian Upgraded
Upgrades the look of the **Utilitarian** mod to be more vanilla-like

### Vending Block: Restocked - Mandala's GUI
Stylizes the GUI of **my *Vending Block: Restocked*** mod to match **Mandala's** GUI style

<sub>(Links to each referenced project can be found in the resource pack's README file)</sub>

## readme-formatting.py
This repository includes a Python script, `readme-formatting.py`, which is used to format the README files for use on CurseForge and Modrinth, as each platform accepts different HTML features.
