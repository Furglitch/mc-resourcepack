This resource pack is a grab-bag of tiny fixes for a few mods, things you barely notice until it isn't there. Mostly small text tweaks and subtle visual nudges to make things read and look a touch clearer in-game.

---

### **Supported Mods**
- [Arda's Sculks](https://modrinth.com/mod/ardas-sculks)
- [Compact Machines](https://www.curseforge.com/minecraft/mc-mods/compact-machines)

### **Changes**
- *Arda's Sculks*
  - Fix 'Ancient Magma' being named 'Deep Granulite Diamond Ore'
  - Add missing apostrophe to Mobs creative tab name
  - Fix 'Ores' creative tab name using 'Arda's Minecraft' instead of 'Arda's Sculks'
- *Compact Machines*
  - Change name of 'Personal Shrinking Device' to 'Player Compression Device' for clarity with Shrink mod.
